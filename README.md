# Concentration Game

## How to start
Compile the .java files in a folder with the images folder.
Run Driver.Java

## Instructions

To play just click a tile and try to find the match to the image that is revealed.
If you reveal two tiles that dont match you will get 3 seconds before they are
covered again. If you reveal two images that do match they will remain uncovered
until the reset button is pressed. Try to reach 8 points in as few moves as possible.
